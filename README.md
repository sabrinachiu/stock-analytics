MarketInsight Pro is a high-performance, real-time stock analytics dashboard built with Python and Streamlit. It leverages the `yfinance` library to provide comprehensive market data, interactive charts, and fundamental analysis.

## Features
- **Real-Time Data**: Fetch latest stock prices and metrics.
- **Interactive Candlestick Charts**: Powered by Plotly for deep price action analysis.
- **Key Statistics**: Market Cap, P/E Ratio, Dividend Yield, and more.
- **Company Profile**: In-depth business summary and industry classification.
- **Historical Data**: Full access to historical pricing tables.

## Technologies Used
- **Streamlit**: Web interface and dashboard layout.
- **yfinance**: Financial data retrieval.
- **Pandas**: Data manipulation and processing.
- **Plotly**: Advanced interactive data visualization.

## How to Run Locally
1. Clone this repository:
   ```bash
   git clone <repo-url>
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   streamlit run app.py
   ```

## Deployment
This app is designed to be easily deployed on **Streamlit Community Cloud**. 
1. Push this code to a public GitHub repository.
2. Log in to [Streamlit Cloud](https://share.streamlit.io/).
3. Connect your GitHub account and select this repository.
4. Click 'Deploy'.
